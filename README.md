## Project: EthioClimate Analytics for COP32

This repository contains climate data analysis for Ethiopias preparation for COP32.

### Setup
-python 3.12
-Virtual envirenment: venv

### Contents
- \'data/\': Raw and processed climate data
- \'notebooks/\': Jupyter notebooks for analysis
- `scripts/`: Python analysis scripts
- `reports/`: Final position papers
- `figures/`: Generated visualizations

### Negotiation-Grade Analysis
This project answers three questions:
1. What is changing? (trend + uncertainty)
2. What did it cause? (impact statistics)
3. What does it demand? (policy asks)
